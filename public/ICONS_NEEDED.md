# PWA Icons Needed

Place the following icon files in this `public` folder:

## Required Icons:

1. **pwa-192x192.png**
   - Size: 192x192 pixels
   - Purpose: Standard PWA icon (Android)

2. **pwa-512x512.png**
   - Size: 512x512 pixels  
   - Purpose: High-res PWA icon (Android, maskable)

3. **apple-touch-icon.png**
   - Size: 180x180 pixels
   - Purpose: iOS home screen icon

4. **favicon.ico**
   - Size: 32x32 pixels (or 16x16, 32x32, 48x48 multi-size)
   - Purpose: Browser favicon

## Generate Icons:

Use one of these tools to generate all sizes from your logo:

- **PWA Asset Generator:** https://www.pwabuilder.com/imageGenerator
- **Favicon Generator:** https://realfavicongenerator.net/
- **PWA Builder:** https://www.pwabuilder.com/

## Design Tips:

- Use a simple, recognizable symbol or logo
- Ensure good contrast for both light and dark backgrounds
- Test on both iOS and Android devices
- Square design works best (avoid text that might be cropped)
